### `Run Locally`

+ Clone the project:

      git clone https://github.com/satishshinde2048/MinishopApp.git  


+ Navigate to the project directory:

      cd Flashcard-Generator-project
                                                                                                       
+ Install dependencies:                                                                                                    

       npm install 
                                                                                                                                                                                                                
+ Start the server:                                                                                                        

       npm start
